# Changelog

All notable changes to the **open-templates organization meta-repository** (`.github`) are documented here.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

Individual template repositories maintain their own changelogs.

## [0.1.0] - 2026-07-04

### Added

- Organization profile catalog in `profile/README.md` with four template categories
- **Repository templates:** [github-repo-template](https://github.com/open-templates/github-repo-template)
- **Package templates:** [npm-package-template](https://github.com/open-templates/npm-package-template)
- **Frontend templates:** [react-supabase-auth-template](https://github.com/open-templates/react-supabase-auth-template)
- **Backend templates:** [cf-hono-supabase-api-template](https://github.com/open-templates/cf-hono-supabase-api-template)
- Supabase auth **bundle** callout (frontend + backend pair)
- Org meta-repo `README.md`, `INSTRUCTIONS.md`, and community docs scaffold
